## Project Description
Heal Together is a web-based platform designed for people dealing with similar health conditions or diseases.
Users can enroll based on their health condition and chat anonymously with others who share the same experience.
The platform promotes emotional support, sharing experiences, and building a supportive community in a safe, private environment.

## Features
- User registration and enrollment based on disease type
- Anonymous chatting with users having similar conditions
- Topic-based chat groups for specific health concerns
- Simple and intuitive interface for easy communication
- Secure handling of user data and interactions

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL

## Project Structure
index.php          -> Main entry point
register.php       -> User registration
login.php          -> User login
chat.php           -> Chat interface for enrolled users
css/               -> Stylesheets
js/                -> JavaScript files
backend/           -> PHP scripts for server-side logic
database/          -> MySQL database schema and configuration
assets/            -> Images, icons, or media files

## How to Run the Project
1. Extract the ZIP file.
2. Place the project folder in your local server directory (e.g., htdocs for XAMPP).
3. Start your local server (Apache & MySQL).
4. Import the database schema using phpMyAdmin.
5. Open a web browser and go to:
   http://localhost/<project-folder-name>
6. Register or login, select your health condition, and start chatting with people having the same condition.
